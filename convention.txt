CONVENTIONS

Titres
------

A l'intérieur d'un chapitre, on utilisera des titres de niveau 2 ou supérieur,
le niveau h1 étant réservé pour le titre de chapitre.

Typographie
-----------

La fonction 'smart punctuation' est limitée pour le français.
Il importera donc d'insérer en caractères UTF8 les guillemets français ainsi que
les espaces insécables.


Noms de logiciels, packages et fonctions
----------------------------------------

Logiciel : le nom sera indiqué en gras.
Exemple : **Word**

Package R : le nom sera indiqué en gras, dans une balise strong avec une classe CSS égale à "package".
Exemple : <strong class="package">questionr</strong>

Fonction : le nom sera indiqué dans une balise code (via le raccourci markdown ``).
Le package dont est issu cette fonction sera précisé dans un attribut data-package.
Le cas échéant (pour une méthode par exemple), on précisera dans un attribut data-rdocumentation le nom
utilisé sur http://www.rdocumentation.org
Exemples :
`freq`{data-package="questionr"}
la méthode `summary`{data-package="prevR" data-rdocumentation="summary-prevR-method"}
pour les objets de la classe `prevR`{data-package="prevR" data-rdocumentation="prevR-class"}

Nom de variable : en italique

Argument d'une fonction :  balise code ou ``
Exemple : l'argument `object`

Valeur manquante : `NA`

Encadrés
--------

Les encadrés sont précisés avec une balise div et la classe CSS appropriée. Les différentes classes disponibles sont :

- info (bleu)
- note (gris)
- astuce (vert)
- important (jaune)
- attention (rouge)

Figures
-------

Elles doivent être présentées au sein de balises <figure>.
Le titre de la figure est précisé avec une balise <figcaption>.
La numérotation est automatique.

Concepts
--------

Sont indiqués avec <dfn> ... </dfn>.
Permettent de produire un index des concepts.

On peut contrôler l'entrée d'index avec
<dfn data-index="package">packages</dfn>

Pour saisir une entrée secondaire, on utilisera un em vide
<dfn data-index="entrée secondaire"></dfn>

On utilisera <dfn lang="en">...</dfn> pour spécifier une entrée en anglais.
